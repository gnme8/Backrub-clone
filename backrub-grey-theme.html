BackRub is a "web crawler" which is designed to traverse the web.

Currently we are developing techniques to improve web search engines. We will make varios service available as soon as possible.

BackRub is a research project of the Digital Library Project in the Computer Science Department at Stanford University.

Some Rough Statistics (from august 29th, 1996)

Total content downloaded: 207.022 gigabytes

Total indexable HTML pages which have not been attempted yet: 30.6822 Milion
Total robot.txt excluded: 0.224249 Milion

BackRub is written in Java and Python and runs on several Sun Ultras and Intel Pentiums running Linux.
